## How to Use

### Local Viewing
1. Open `index.html` in a web browser
2. Use arrow keys or click to navigate slides
3. Press 'S' to open speaker notes in a new window
4. Press 'ESC' to see slide overview

### Keyboard Shortcuts
- **Arrow Keys** - Navigate slides
- **Space** - Next slide
- **Shift+Space** - Previous slide
- **S** - Speaker notes
- **F** - Fullscreen
- **ESC** - Slide overview
- **Alt + Click** - Zoom in on content

## Deploying to GitHub Pages

1. Create a GitHub repository named `quarterly-earnings-presentation`
2. Push the files to the repository
3. In repository settings, enable GitHub Pages from the main branch
4. Access your presentation at: `https://[USERNAME].github.io/quarterly-earnings-presentation/`

## Customization

To customize the presentation:
- **Colors:** Modify the CSS custom properties in the `<style>` section
- **Content:** Edit slide content directly in the HTML
- **Theme:** Change the theme CSS from `black.min.css` to other themes
- **Fonts:** Update font-family in the CSS section

## Speaker Notes Usage

Speaker notes are embedded in each slide using the `<aside class="notes">` tag. Access them by:
- Pressing 'S' during presentation
- Opening in standalone speaker view
- Viewing in presentation mode with notes panel
